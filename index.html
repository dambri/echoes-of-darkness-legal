⚔️ Echoes of Darkness

Your Destiny Companion on iOS

A powerful iOS companion app for Destiny players, featuring raid tracking, character informations, and comprehensive activity history powered by the Bungie.net API.

📱 iOS 🎮 Destiny Integration 🔐 OAuth Authentication ⚡ SwiftUI
Download on the App Store

✨ Features

🎯 Raid Tracking

Browse and track raid activities with detailed statistics, achievements, and completion history.

👤 Character Informations

View all your Destiny characters across platforms, including light levels, classes, and equipment.

🌐 Cross-Platform Support

Seamlessly manage characters across Xbox, PlayStation, and other platforms.

📊 Activity History

Comprehensive tracking of your game activities and progression over time.

🔒 Secure Authentication

OAuth integration with Bungie.net ensures secure access to your Destiny data.

🎨 Beautiful UI

Modern glass morphism design with custom Orbitron typography and gradient effects.

🛠 Technology Stack

Swift SwiftUI OAuthSwift Firebase Kingfisher Bungie.net API iOS 15+ Async/Await

Architecture Highlights

  • SwiftUI - Modern declarative UI framework
  • MVVM Pattern - Clean separation of concerns with ViewModels
  • Environment Objects - Centralized state management
  • Swift Concurrency - Async/await for network operations
  • Firebase Crashlytics Integration - Monitoring crash and errors

📱 Key Components

Authentication Flow

Secure OAuth 2.0 authentication with Bungie.net, handling token management and refresh automatically.

Character View

Displays all Destiny characters with emblem backgrounds, class information, race details, and current light level. Supports platform icons for Xbox, PlayStation, and other platforms.

Raid Details

Comprehensive raid information including achievements, activity history, and detailed statistics for each raid encounter.

Dashboard

Central hub for accessing all app features with an intuitive tab-based navigation system.

🎨 Design Philosophy

Echoes of Darkness embraces a dark, space-themed aesthetic that matches the Destiny universe. The app features:

  • Custom Orbitron font for a futuristic feel
  • Glass morphism effects for modern UI elements
  • Nebula-inspired gradient backgrounds
  • Smooth animations and transitions
  • Native platform integration with SF Symbols
