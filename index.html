⚔️ Echoes of Darkness

Your Destiny Companion on iOS

A powerful iOS companion app for Destiny players, featuring raid tracking, character management, and comprehensive activity history powered by the Bungie.net API.

🚀 Getting Started

Prerequisites

  • Xcode 14.0 or later
  • iOS 15.0 or later
  • Bungie.net API credentials
  • Firebase project (for backend services)

Installation

  1. Clone the repository:
    git clone https://github.com/yourusername/echoesOfDarkness.git
  2. Open echoesOfDarkness.xcodeproj in Xcode
  3. Configure your Bungie.net API credentials
  4. Add your Firebase configuration file (GoogleService-Info.plist)
  5. Build and run the project

Configuration

You'll need to:

  • Register your app with Bungie.net to obtain OAuth credentials
  • Set up Firebase project for backend services
  • Configure the OAuth redirect URL in your app settings
